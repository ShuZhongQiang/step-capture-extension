# 步骤图录制器插件

一个类似于ScribeHow/Tango的浏览器步骤图录制插件，用于记录用户操作并生成详细的步骤指南。

## 功能特性

- 📹 **录制控制**：开始/停止录制按钮
- 🖱️ **点击记录**：自动捕获点击操作
- 🔍 **元素信息**：获取元素选择器和文本内容
- ✨ **高亮效果**：点击时显示红色边框高亮
- 📷 **自动截图**：每次点击自动截取当前页面
- 📝 **步骤管理**：查看和管理已记录的步骤
- 📤 **导出功能**：支持导出为Markdown和HTML格式

## 安装方法

1. 克隆或下载本仓库到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本仓库的根目录
6. 插件安装完成，会在浏览器工具栏显示

## 使用方法

1. 点击浏览器工具栏中的插件图标，打开弹出面板
2. 点击「开始录制」按钮
3. 在网页上进行操作，每次点击都会被记录
4. 点击「停止录制」按钮结束录制
5. 在步骤列表中查看已记录的步骤
6. 选择导出格式（Markdown或HTML），点击相应按钮导出

## 技术实现

- **前端**：HTML + CSS + JavaScript
- **浏览器扩展技术**：Chrome Extension API
- **存储**：localStorage
- **截图功能**：html2canvas库
- **导出格式**：Markdown和HTML

## 注意事项

- 插件需要「活动标签页」和「存储」权限
- 截图功能可能会受到跨域限制
- 录制过程中会轻微影响页面性能
