# 2026-04-10 工作日志

## Bug 修复 - getActiveTab 三重 fallback

- 问题：侧边栏打开时焦点窗口是扩展窗口，`lastFocusedWindow:true` 拿不到用户网页 tab
- 修复：改用三重策略 fallback（1.聚焦窗口active tab → 2.所有窗口active tab → 3.最近访问的网页tab），`isWebTab()` 过滤允许 `http://`、`https://`、`file://`

## 高亮样式改为新拟态风格

- 颜色从生硬红色 `#ef4444` 改为柔和蓝紫 `rgba(99, 102, 241, ...)`
- 圆角从 4px 加宽到 12px
- 阴影改为三层柔和光晕叠加 + 呼吸动效（1.8s ease-in-out）

## 浏览器插件优化 - Side Panel 侧边栏改造

- 点击扩展图标 → 浏览器右侧打开固定面板，不因点击网页内容而关闭

## 浏览器插件优化 - 步骤清空功能

- 项目：步骤图录制器（Chrome 浏览器插件，MV3）
- 修改文件：`popup.html`、`popup.js`、`styles.css`
- 功能说明：
  1. 在步骤列表标题右侧新增"🗑 清空"按钮（`id="clear-steps"`）
  2. 点击清空按钮弹出确认对话框，确认后清空 `chrome.storage.local` 中的 `steps` 数组
  3. 点击"开始录制"时**自动清空**上次录制的步骤，无需手动操作
  4. 新增 `.steps-header` 弹性布局样式和 `.clear-button` 悬停变红样式
